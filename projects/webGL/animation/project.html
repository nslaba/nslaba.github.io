WebGL Animation

Hierarchical character animation built with WebGL and custom GLSL shaders.

Overview

Characters constructed as hierarchical objects using WebGL's primitive shapes. The scene uses custom fragment shaders for dynamic visual effects, with textures derived from my own paintings.

WebGL GLSL JavaScript Hierarchical Modeling

Features

The animation includes a 360-degree camera flyaround synchronized with real-time parameters - light and color shift based on elapsed time. The psychedelic aesthetic combines technical constraints with playful visual experimentation.
