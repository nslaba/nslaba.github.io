Hermite Splines

Smooth curves are everywhere in computer graphics. This project explores Hermite splines with arc-length parameterization for consistent animation timing.

Overview

Hermite splines give you precise control over curve shape through control points and tangent vectors. They're essential for keyframe animation, path interpolation, and designing complex surfaces.

C++ OpenGL Catmull-Rom Quaternions

Features

  • 40 control points with interactive manipulation via console commands
  • Catmull-Rom tangent generation with second-order accurate boundary conditions
  • Arc-length parameterization using piecewise linear approximation and lookup table
  • File I/O for saving and loading spline configurations
  • Quaternion-based orientation alignment to keep objects tangent to the path
  • Ease-in/ease-out motion at path endpoints with constant speed in between

The Math

The trickiest part is arc-length parameterization. Without it, objects speed up and slow down unpredictably as they traverse the curve. By building a lookup table that maps distance along the curve to parameter values, you get uniform sampling: P(u(s(t))).

Orientation is handled with quaternions to avoid gimbal lock and ensure smooth rotation as the object follows the path. The car's major axis stays aligned with the tangent vector at each point.
