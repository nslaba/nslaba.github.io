OpenGL Hermite Spline Project

Introduction to OpenGL and Hermite Splines

I have experience with older versions of OpenGL. A fun project was building a Hermite Spline system and a simulator.

Hermite splines are an essential tool in computer graphics for creating smooth curves and animations. They allow for precise control over the curve's shape by manipulating the control points and tangents. This capability makes Hermite splines invaluable for tasks such as keyframe animation, path interpolation, and even in the design of complex shapes and surfaces.

Hermite Spline Project

The task was to build a Hermite Spline system and a simulator that meets the following criteria:

  • Hermite spline class with a maximum of 40 control points.
  • Catmull-Rom initialization with second-order accurate boundary conditions using the command: system cr
  • Manipulation of control points and tangents using the console commands.
  • Arc Length parameterization using a piecewise linear approximation and lookup table: system getArcLength
  • Input/output from/to a file: system load "" and system export ""
  • Loading the path for the object: system objectpath load ""
  • Aligning the major axis of the object with the tangent of the path as it moves.
  • Ensuring the moving object starts and stops in an easy-in, easy-out way, maintaining a constant speed during the rest of the motion.
